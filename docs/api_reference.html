pyLD.create_surface(pitch, volume, PSD=None, num_smoothing=30, method_smoothing='humphrey')

Create a smooth surface mesh of a volume. Smoothing is based a humphrey filter, and surface area per face is calcudated. Faces within PSD regions are labeled (Note, currently BEFORE smoothing).

Parameters

  • pitch (float) – Unit length per voxel

  • volume (numpy) – Input volume (3D array, bool preferred)

  • PSD (numpy) – PSD volume (3D array)

  • num_smoothing (int) – Number of smoothing rounds for the target surface mesh method_smoothing(str): Smoothing method: ‘laplacian’, ‘humphrey’, ‘mut_dif_laplacian’, or ‘taubin’

Returns

Tuple containing:

  • smooth_vertices (numpy[float]): Vertices of smoothing mesh (3xX array)

  • smooth_faces (numpy[int]): Faces of smoothing mesh (3xY array)

  • smooth_area_per_face (numpy[float]): Areas of faces (Y array)

  • surface_areas (numpy[float]): Surface areas in voxel space (3D array)

  • id_face_psd (numpy[bool]): Faces that are located at PSD (X array, bool)

Return type

(tuple)

pyLD.get_labeled_concs(lm_filename, labels, output_filename=None, monitor_species='Ca')

Get time series of moleuclar numbers/concentrations within labeled volumes from LM simulation result.

Parameters

  • lm_filename (str) – Filename of lm simulation.

  • labels (numpy[int/bool]) – Label volume (3D array). id=0 will be ignored.

  • output_filename (none/str) – If None, get_labeled_concs will give returns below. If specified, it will save a hdf5 file.

  • monitor_species (none/str) – If None, get_labeled_concs will give no messages in the console. If a moleuclar species is specified, it will show a example result.

Returns

Tuple containing:

  • num_molecules (dict): Time series of the numbers of molecules of the specified molecular species. The dict container has {‘species1’: [[num_label1_t1, num_label2_t1, …], [num_label1_t2, num_label2_t2, …], …], ‘Species2’: [[num_label1_t1, num_label2_t1, …], [num_label1_t2, num_label2_t2, …], …], … }.

  • uMs (dict): Time series of molecular concentrations (number per labeled volume, in the unit of uM). The dict container has {‘species1’: [[conc_label1_t1, conc_label2_t1, …], [conc_label1_t2, conc_label2_t2, …], …], ‘Species2’: [[conc_label1_t1, conc_label2_t1, …], [conc_label1_t2, conc_label2_t2, …], …], … }.

  • timepoints (numpy[float]): Timepoints (s)

  • ids_spine (numpy[int]): Labels in the volume. The numpy array has [label1, label2, …].

Return type

(tuple)

  • label_ids (numpy[int]): Labels in the volume. The numpy array has [label1, label2, …].

class pyLD.RepeatRun(template_lm_file=None, lm_option=['-r', '1', '-sp', '-sl', 'lm::rdme::MpdRdmeSolver'], exec_periods=[1], exec_events=[<function null_event>], event_params=None, output_dir='results', output_prefix='', output_num_zero_padding=4, label_volume_file=None)

Repeat simulation runs. Output of each run is set to be a inital state of the next run. Users can modify the initial state (4D array; 3D volume + 16 species space), which can be considered as event function.

Returns

RepeatRun object that contains the following instance variables:

  • ’template_lm_file’ (str): Template lm file

  • ’lm_option’ (list): Optional arguments passing to lm

  • ’exec_periods’ (list[float]): Simulation period in each run

  • ’exec_events’ (list[obj]): Event function to modify the lattice space before each run

  • ’event_params’ (dict/list[dict]/tuple[dict]): Parameters passing to event function

  • ’output_dir’ (str): Directory that stores simulation results

  • ’output_prefix’ (str): Prefix of output lm filenames that stores simulation results

  • ’output_num_zero_padding’ (int): Number of zero padding in output lm filenames

  • ’label_volume_file’ (str): labeled volume file (optional)

Return type

(pyLD.RepeatRun)

Returns: bool

(bool): True if succeeded. Also simulation results are stored in lm files in output_dir.

pyLD.create_surface(pitch, volume, PSD=None, num_smoothing=30, method_smoothing='humphrey')

Create a smooth surface mesh of a volume. Smoothing is based a humphrey filter, and surface area per face is calcudated. Faces within PSD regions are labeled (Note, currently BEFORE smoothing).

Parameters

  • pitch (float) – Unit length per voxel

  • volume (numpy) – Input volume (3D array, bool preferred)

  • PSD (numpy) – PSD volume (3D array)

  • num_smoothing (int) – Number of smoothing rounds for the target surface mesh method_smoothing(str): Smoothing method: ‘laplacian’, ‘humphrey’, ‘mut_dif_laplacian’, or ‘taubin’

Returns

Tuple containing:

  • smooth_vertices (numpy[float]): Vertices of smoothing mesh (3xX array)

  • smooth_faces (numpy[int]): Faces of smoothing mesh (3xY array)

  • smooth_area_per_face (numpy[float]): Areas of faces (Y array)

  • surface_areas (numpy[float]): Surface areas in voxel space (3D array)

  • id_face_psd (numpy[bool]): Faces that are located at PSD (X array, bool)

Return type

(tuple)

pyLD.null_event(lattice, sys_param, event_param)

Null event function in RepeatRun. Any operation will not be executed. Initial state of molelcules can be modified in (lattice; 4D array, 3D volume + 16 species space).

Parameters

  • lattice (numpy[uint8]) – Lattice space (4D array, 3D space plus 16 slots)

  • sys_param (dict) – System parameters that contains - ‘i’ (int): Exec id, - ‘time’ (float): Start time - ‘species’ (dict): Connection between a molecular name and its id - ‘label volume’ (numpy[int]): Label volume if specified (3D array) - ‘label ids’ (numpy[int]): Label ids if specified (1D array)

  • event_param (dict) – It contains user-defined parameters for event function

Returns: Tuple containing:

  • lattice: (numpy[uint8]): Lattice space (4D array, 3D space plus 16 slots)

  • event_param: (dict): User-defined values can be passed to the next event
